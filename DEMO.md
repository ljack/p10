## 📱 UI Layout

The component displays notes in a **responsive grid**:
- **Desktop**: 3-4 cards per row
- **Tablet**: 2 cards per row  
- **Mobile**: 1 card per row

Each **note card** includes:
- **Header**: Title and creation date
- **Body**: Content preview (truncated)
- **Footer**: Last updated info (if different)
